# LevelTone — Languages / Sprachen / Langues / 语言

LevelTone's interface is localized in **30 languages**, each with its own README and a
full user manual. The app auto-selects your device language.

> ⚠️ **Translations are machine-assisted and not reviewed by native speakers.** English is
> the source. Spotted a mistake in your language? Corrections are very welcome — open a
> [PR](../../pulls) or an [issue](../../issues).
